
The file scoresby_sound_utm.shp is cropped from the original OpenStreetMap polygons.
The file updated_scoresby_sound_utm.shp is a modification of these polygons using QGIS based on satellite raster data to improve
 accuracy.
 The files shrunk*_scoresby_soundutm.shp are obtained by applying a geometric buffer of 100, 200 and 300m to the scoresby_sound_utm.shp.
